# Sungho Yun Academic Homepage

This is a static academic portfolio website generated from the provided CV.

## Files

- `index.html` — main homepage
- `style.css` — responsive design and visual styling
- `script.js` — mobile navigation and small UI behavior
- `assets/sungho-yun.jpg` — profile image extracted from the CV

## How to preview locally

Open `index.html` directly in a browser, or run:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## How to publish with GitHub Pages

1. Create a GitHub repository.
2. Upload these files to the repository root.
3. Go to Settings → Pages.
4. Select the main branch and root directory.
5. Save, then open the generated GitHub Pages URL.

## Note

The public page includes email but excludes the phone number from the CV by default for privacy. Add it manually to `index.html` if needed.
